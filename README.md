# Building a RESTful API with Koa and Postgres

Project base...

1. Fork/Clone
1. Install dependencies - `npm install`
1. Sanity check - `npm start`
1. Test - `npm test`

### Source build base on https://mherman.org/blog/building-a-restful-api-with-koa-and-postgres/

1. Knex v0.13.0
1. pg v7.1.2
1. koa-router v7.2.1
